Press

Test utility function.

Fires a key press on an element in the browser for testing purposes.

   R←[X] Press Y

X is an optional element. If omitted, it defaults to the current test element.

Y is a JavaScript key specification, for example:

'Enter'
'Ctrl' 'Shift' 'ArrowRight'
'Alt' 'F7'
 
`R` is `0`.
