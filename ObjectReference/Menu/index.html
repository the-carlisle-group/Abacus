Menu Object

Description

A component for creating menu bars, hamburger menus, popup menus, and drop down menus.

Methods

NewDropdownMenuNewHamburgerMenuNewItemNewMenuBarShow

Commentary

The Menu allows you to create a menu and then display it as a popup menu, a menu bar, a hamburger menu, or a drop down menu.

The New method create a new menu or submenu.

The NewItem method creates a new menu item.
