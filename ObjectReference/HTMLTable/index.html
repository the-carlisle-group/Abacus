HTMLTable Object

Methods

Cells

The HTMLTable object provides functionality for creating and manipulating HTML table elements. It is an HTMLElement object whose Tag property is table, so all methods and properties that apply to HTMLElement apply also to HTMLTable.

Tables are created with the NewTable method:
