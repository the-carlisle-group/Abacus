# Input Components

Abacus provides 5 basic input components: [TextInput](), [NumberInput](),
[DateInput](), [DropList](), and [CheckBox](). In addition, the [FileInput]()
component is a thin cover on TextInput for specifying a file name.

Each input component has a `Value` property that may be specified when using `New`, and
subseqently queried directly. Each input component has an `InitValue` method and a `SetValue`
method. The `InitValue` method is used to specify the value on an instance in a unconnected
