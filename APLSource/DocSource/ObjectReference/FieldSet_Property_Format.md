# FieldSet Format Property

Specifies the alignment of labels and fields.

|Value|Description|
|:=====|:===========|
|`stacked`    | Vertical layout with labels over fields|
|`inline`     | Vertical layout wth labels to the left of fields|
|`stacked-row`| Horizontal layout with labels over fields|
|`inline-row` | Horizontal layout with labels to the left of fields|
