# HTMLDocument NewDocument Method

Creates a new HTML Document.

~~~
     R←NewDcocument X
~~~

`X` is <body> content.
R is a new HTMLDocument.

## Examples:

~~~
      D←A.NewDocument ''
      A.Render D
<html lang="en">
  <head>
    <meta charset="utf-8"></meta>
  </head>
  <body></body>
</html>
~~~
