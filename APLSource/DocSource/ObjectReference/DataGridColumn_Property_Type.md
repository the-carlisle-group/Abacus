# DataGridColumn Type Property

The column type. Current supported types are Char, VarChar, Int, Float, Date,
Dec[1,2,3], Select, MultiSelect, and CheckBox. 
If the type is provided and the Value property is provided, they must correspond.
If no type and no value is provided, the type defaults to VarChar.
If no type is provided but a Value property is provided, the type is inferred from the value.
