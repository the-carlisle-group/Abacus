# HTMLDocument KeyDown Event

The event message space supplied as a right argument
to the callback function contains:

|Name      |Value|
|:=========|:=====|
|Event     | `KeyDown`               |  
|Component | The Document instance   |
|Document  | The Document instance   |
|Key       | The fully qualified key | 

Menu items or buttons or other elements with associated shortcuts
take precedence.
