# DataGrid RepaceCellValue Method

Replaces a single value in the datagrid.

~~~

~~~

`G` is the grid instance. `V` is a value. `R` is the row index
of the replacment cell. `C` is either the column index or the column name
of the replacement cell.

See also [ReplaceCellValues]() for replacing a range of cells.
