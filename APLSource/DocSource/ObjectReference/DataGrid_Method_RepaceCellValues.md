# DataGrid RepaceCellValues Method

Replaces a matrix of values in the datagrid.

~~~
R‚ÜêG ReplaceCellValues M (R C)
~~~

`G` is the grid instance. `M` is a matrix of values. `R` is the row index
upperleft corner to begin replacment. `C` is either a column index or a column name
of the the upperleft corner to begin replacement.

See also [ReplaceCellValue]() for replacing a single cell value.
